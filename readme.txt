== Description ==

Cascading Stylesheet (CSS) editor (WYSIWYG/Context) written in JavaScript and PHP

== Installation ==

1. Write `/jsCSSEditor/` to the `/wp-content/plugins/` directory
2. Activate the jsCSSEditor plugin through the 'Plugins' menu in  WordPress url  /wp-admin/plugins.php
3. holds CTRL and click the left button of mouse.
